vs2019 快捷键。

> 在调 试 程序 或 运行 程序 的 过程中， 通常 可用 以下 4 种 快捷键 来 操作。 （1） 按 F5 快捷键 实现 调试 运行 程序。 （2） 按 Ctrl+ F5 快捷键 实现 不调 试运行 程序。 （3） 按 F11 快捷键 实现 逐 语句 调试 程序。 （4） 按 F10 快捷键 实现 逐 过程 调试 程序。
